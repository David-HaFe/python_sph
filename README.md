# python_sph
Python implementation of SPH method. Purely didactic

## dependencies
In order to run this code, you will need a virtual environment
```
python -m venv .venv
```
and activate it
```
source .venv/bin/activate
```
Once this is done, you will need to install
* numpy
* matplotlib
* scipy
```
pip install numpy
pip install matplotlib
pip install scipy
```
